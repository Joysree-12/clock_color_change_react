⏰ Clock Color Change App (React)

This is a simple React application that displays a live clock and allows the user to change the color of the clock dynamically using a dropdown menu.

✅ Features

Live updating digital clock (updates every second)

Dropdown menu to select a color

The selected color is applied only to the clock time and also shown in the “Color: ...” line

📦 Technologies Used

React (Functional Components)

React Hooks → useState, useEffect
